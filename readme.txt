Embed a PopSurvey wherever you'd like with a shortcode.

== Description ==

The PopSurvey Shortcode Plugin lets you easily embed a <a href="http://popsurvey.com">PopSurvey</a> right in your WordPress site.

Example: `[popsurvey s="aw15ut"]`

You can get your shortcode from the Promote area of your PopSurvey.

== Installation ==

1. Upload `popsurvey.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Booyah.

You can also do a search for "PopSurvey Shortcode Plugin" in the admin area of your WordPress site from Plugins < Add New.
